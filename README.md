# @snguesssan/create-pi-ag-ui

Scaffold a new **Pi AG-UI** project — Next.js 15 + React 19 + CopilotKit + Pi SDK.

## Usage

```bash
npx @snguesssan/create-pi-ag-ui my-app
```

## Options

| Flag | Description |
|------|-------------|
| `--pm npm\|pnpm\|yarn\|bun` | Choose package manager |
| `--skip-install` | Skip automatic dependency installation |
| `-h, --help` | Show help |

## Examples

```bash
# Interactive (prompts for name and package manager)
npx @snguesssan/create-pi-ag-ui

# With project name
npx @snguesssan/create-pi-ag-ui my-app

# With package manager
npx @snguesssan/create-pi-ag-ui my-app --pm pnpm

# Skip install
npx @snguesssan/create-pi-ag-ui my-app --skip-install
```

## What you get

- Streaming chat interface with CopilotKit
- 8 frontend tools (notifications, theme toggle, bookmarks, data tables...)
- Human-in-the-loop interactions (confirmations, user input)
- Agent state visualization (status, timeline, metrics)
- AI model switching and extended thinking support
- Tailwind CSS styling with dark mode
